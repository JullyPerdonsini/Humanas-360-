# EduPlanner EM

Aplicativo web estático para apoiar professores do Ensino Médio na criação rápida de materiais.

## Recursos
- Tela inicial com visual simples e amigável.
- Seleção de disciplina e ano/série.
- Campo de tema/conteúdo e objetivo da aula.
- Geração de atividade, questões adaptadas e texto simplificado.
- Histórico local de materiais gerados.
- Download do conteúdo gerado em `.txt`.
- Biblioteca de materiais gratuitos com filtro por disciplina.
- Personalização rápida (nome da escola, temas de cor e alto contraste).

## Executar localmente

```bash
python3 -m http.server 8080
```

Acesse: `http://localhost:8080`.
